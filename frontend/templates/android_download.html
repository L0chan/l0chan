Android Download

The Android APK file has not been built yet, so there is no direct APK download available in this project folder.

For mobile install without an APK, Android Chrome needs the app to open from HTTPS. Local links like http://192.168.x.x:5000 usually do not show Install App.

Use one of these options:

  1. Open the app through an HTTPS tunnel or hosted website, then tap Install App.
  2. Build an APK with PWABuilder after hosting the app on HTTPS.
  3. Place the APK at release/NearbyPriceFinder-Android.apk and this button will become a real download.
